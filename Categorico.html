El análisis de las medias muestra que la mayoría de los atributos de calidad son poco frecuentes, lo que implica distribuciones altamente desbalanceadas. Este comportamiento explica por qué ciertas visualizaciones como los boxplots o las gráficas de densidad no resultan informativas, ya que la mayor parte de los valores se concentran en cero.

Por esta razón, las visualizaciones más adecuadas para este tipo de datos son los gráficos de conteo y las tablas de contingencia, que permiten observar directamente la frecuencia de ocurrencia de cada categoría en relación con la variable objetivo.

Dado que las variables son binarias, el coeficiente de correlación de Pearson es equivalente al coeficiente φ (phi), lo que lo convierte en una medida adecuada para evaluar asociaciones entre pares de variables.

El análisis de correlación se realiza tanto entre las características como entre cada característica y la variable objetivo. Este procedimiento permite identificar posibles redundancias entre atributos y evaluar qué características presentan una mayor asociación con la clasificación de los requisitos.

Aunque las variables son binarias, el factor de Fisher sigue siendo válido, ya que no mide dispersión continua sino la capacidad de una característica para separar clases. En el caso de variables binarias, la media representa una proporción y la varianza está determinada por dicha proporción, lo que permite cuantificar la separabilidad entre clases.

    'Availability (A)',
    'Fault Tolerance (FT)',

    'Look & Feel (LF)',
    'Maintainability (MN)',
    'Operability (O)',
    'Performance (PE)',
    'Portability (PO)',
    'Scalability (SC)',
    'Security (SE)',
    'Usability (US)',
    'Other (OT)'
]

alpha1, alpha2 = 0.5, 0.5

# -----------------------------
# Fisher según TU definición
# -----------------------------
def get_fisher(feature_name):
    overall_mean = df[feature_name].mean()
    bgv = 0  # Between-group variance
    wgv = 0  # Within-group variance
    
    for c in df['IsQuality'].unique():
        subset = df[df['IsQuality'] == c][feature_name]
        n_c = len(subset)
        bgv += n_c * (subset.mean() - overall_mean) ** 2
        wgv += n_c * subset.var() if n_c > 1 else 0
        
    return bgv / wgv if wgv != 0 else 0

step = 2

while remaining:
    print(f"\n--- PASO {4 if step == 2 else 6}: Calculando s{step} ---")
    scores_step = {}
    k = len(selected) + 1
    
    for j in remaining:
        # Correlación promedio con las ya seleccionadas
        corrs = [abs(df[j].corr(df[s])) for s in selected]
        avg_corr = sum(corrs) / (k - 1)
        
        relevance = alpha1 * f_scores[j]
        redundancy = alpha2 * avg_corr
        total = relevance - redundancy
        
        scores_step[j] = total
        
        print(f"Evaluando '{j}':")
        print(f"  > Relevancia (Fisher): {relevance:.6f}")
        print(f"  > Redundancia (Corr promedio con {selected}): {redundancy:.6f}")
        print(f"  > PUNTAJE TOTAL: {total:.6f}")
    
    best_j = max(scores_step, key=scores_step.get)
    print(f" GANADOR para s{step}: '{best_j}'")
